# 🌐 Loopstudios VR Landing Page

A modern, fully responsive landing page for a virtual reality development studio. This project demonstrates responsive web design using **HTML5**, **Tailwind CSS**, and **Vanilla JavaScript**, with a strong focus on clean layouts, responsive imagery, smooth animations, and an engaging user experience.

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Rizwan-46/loopstudios-tailwind-css.git

cd loopstudios-tailwind-css
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start Tailwind CSS Watch Mode

```bash
npx tailwindcss -i ./src/input.css -o ./src/output.css --watch
```

If your `package.json` includes custom scripts, you can also run:

```bash
npm run dev
```

or

```bash
npm run watch
```

### 4. Open the Project

Open `src/index.html` directly in your browser or launch it using the **Live Server** extension in Visual Studio Code for automatic reloading during development.
